## Choosing the Right Tool

<AccordionGroup>
  <Accordion title="Best for beginners">
    **Loop** and **Rectangle** are excellent choices for beginners due to their simple interfaces and intuitive controls. Loop's radial menu provides visual feedback that makes learning window management easier.
  </Accordion>
  
  <Accordion title="Best for power users">
    **Loop**, **Hammerspoon**, **Yabai**, and **BetterTouchTool** offer the most flexibility for power users. Loop combines ease of use with advanced features like cycles and automation.
  </Accordion>
  
  <Accordion title="Best for trackpad gestures">
    **Swish**, **BetterTouchTool**, **Rectangle Pro**, and **Multitouch** excel at trackpad gesture support. Loop focuses on modifier + trackpad rather than complex gestures.
  </Accordion>
  
  <Accordion title="Best for tiling window managers">
    **Yabai**, **Amethyst**, and **AeroSpace** are dedicated tiling window managers for users who prefer automatic window layouts. Loop focuses on manual control with visual feedback.
  </Accordion>
  
  <Accordion title="Best for automation">
    **Loop**, **Hammerspoon**, **Yabai**, **BetterTouchTool**, and **Rectangle Pro** offer the most automation capabilities. Loop's URL scheme makes it easy to integrate with external tools.
  </Accordion>
  
  <Accordion title="Best value">
    **Loop**, **Rectangle**, **Hammerspoon**, **Yabai**, **Amethyst**, **AeroSpace**, and **MacsyZones** are all completely free. Loop stands out as the only free option with a radial menu and extensive theming.
  </Accordion>
</AccordionGroup>

## macOS Native Window Management

macOS 15 (Sequoia) introduced native window tiling features:

<Info>
  macOS 15+ includes basic window tiling via drag-to-edge snapping and keyboard shortcuts. However, it lacks many features found in Loop such as:
  - Radial menu interface
  - Custom window positions
  - Action cycles
  - Stashed windows
  - Theming options
  - Advanced automation
</Info>

Loop complements or replaces macOS native window management with more control and customization.

## Detailed Comparisons

<AccordionGroup>
  <Accordion title="Loop vs Rectangle">
    **Similarities:**
    - Both free and open source
    - Keyboard shortcuts for window management
    - Preview windows
    - Drag to snap
    - Basic action cycles
    
    **Loop advantages:**
    - Radial menu for mouse/trackpad control
    - Extensive theming options
    - Custom window frames
    - Stashed windows
    - More customization overall
    
    **Rectangle advantages:**
    - More mature/stable (longer development history)
    - Simpler interface for keyboard-only users
  </Accordion>
  
  <Accordion title="Loop vs Rectangle Pro">
    **Rectangle Pro** is the paid version of Rectangle ($9.99).
    
    **Loop advantages:**
    - Completely free (Rectangle Pro costs $9.99)
    - Radial menu
    - Theming options
    
    **Rectangle Pro advantages:**
    - Trackpad gestures
    - Save/restore workspaces
    - Resize adjacent windows
    - More polish and refinement
  </Accordion>
  
  <Accordion title="Loop vs Magnet">
    **Magnet** costs $4.99.
    
    **Loop advantages:**
    - Free vs $4.99
    - Open source
    - Radial menu
    - Theming options
    - Custom frames
    - Stashed windows
    - Action cycles
    - More keyboard shortcuts
    
    **Magnet advantages:**
    - Trackpad gestures
    - Simpler interface
  </Accordion>
  
  <Accordion title="Loop vs BetterTouchTool">
    **BetterTouchTool** costs $14.00 and is a comprehensive input customization tool.
    
    **Loop advantages:**
    - Free vs $14.00
    - Focused on window management
    - Radial menu
    - Theming options
    - Simpler to configure for basic window management
    
    **BetterTouchTool advantages:**
    - Extremely comprehensive customization
    - Trackpad gestures
    - Keyboard shortcuts beyond window management
    - Touch Bar support
    - Mouse gestures
    - Pin windows on top
  </Accordion>
  
  <Accordion title="Loop vs Hammerspoon">
    **Hammerspoon** is free, open source, and script-based.
    
    **Loop advantages:**
    - GUI configuration (no Lua scripting required)
    - Radial menu
    - Theming options
    - Preview window
    - Easier for non-programmers
    
    **Hammerspoon advantages:**
    - Unlimited customization via Lua scripting
    - Can control entire macOS system, not just windows
    - Save/restore workspaces
    - More powerful for advanced users comfortable with coding
  </Accordion>
  
  <Accordion title="Loop vs Yabai">
    **Yabai** is free, open source, and a tiling window manager.
    
    **Loop advantages:**
    - GUI configuration
    - Radial menu
    - Preview window
    - Theming options
    - Easier to learn
    - Manual control over window placement
    
    **Yabai advantages:**
    - Automatic tiling layouts
    - Pin windows on top
    - More suited for keyboard-driven workflows
    - Resize adjacent windows automatically
  </Accordion>
</AccordionGroup>

- **Ease of use** with the radial menu and visual feedback
- **Power and flexibility** with keyboard shortcuts, cycles, and automation
- **Customization** with extensive theming options
- **Value** as a completely free, open-source solution
