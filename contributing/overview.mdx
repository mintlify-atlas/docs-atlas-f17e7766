Welcome to Loop! We're excited that you're interested in contributing to this project. Whether you're fixing bugs, adding features, creating icons, or translating to new languages, your contributions help make Loop better for everyone.

## Areas of Focus

There are many ways you can contribute to Loop:

1. **Add translations** - Help make Loop accessible to more people by adding your language or fixing translation errors
2. **Code improvements** - Submit new features, refactor existing code, or fix bugs (make sure to open an issue first)
3. **Icon design** - Create beautiful icons that capture the feel of Loop
4. **Bug reports** - Found a bug? Report it in the issues tab and help us improve Loop's stability

## AI Usage Policy

Loop has a strict policy regarding AI-assisted contributions. Before contributing, please familiarize yourself with our approach:

- **All AI usage must be disclosed** - State the tool you used (Claude, ChatGPT, GitHub Copilot, Cursor, etc.) and how it assisted your work
- **Human verification required** - You are responsible for ensuring AI-suggested changes actually work. All AI-generated code must be manually reviewed and tested
- **No AI-generated media** - Only text and code are acceptable AI-generated content
- **Be prepared to explain** - You must understand and be able to explain any code or text you submit

For complete details, see the [AI Usage Policy](https://github.com/MrKai77/Loop/blob/main/AI_POLICY.md) in the repository.

<Note>
This policy exists to maintain code quality and protect maintainers' time. AI is welcome as a tool, but contributors remain fully responsible for output quality and correctness.
</Note>
